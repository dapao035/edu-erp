# 小猿管家培训教育行业ERP系统

#### 介绍
小猿管家培训教育行业ERP系统，包括api端，管理端，小程序端，暂未开发其他端，如有需要自行开发。
支持多组织管理，每个组织有分校，可以为每个组织分配二级域名访问。
wepy开发的小程序端，学员使用。
vue+elementui开发的管理端
php+mysql开发的api端
